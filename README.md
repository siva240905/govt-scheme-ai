# 🤖 Govt Scheme AI

Govt Scheme AI is an intelligent web application that helps users discover government welfare schemes based on their personal profile and eligibility. The platform simplifies the process of finding suitable schemes through AI-powered recommendations and an intuitive user interface.

## ✨ Features

- 🤖 AI-Powered Scheme Recommendations
- 🔍 Smart Search & Filtering
- ✅ Eligibility Checking
- 📋 Detailed Scheme Information
- 📂 Category-wise Scheme Browsing
- ⚡ Fast & Responsive User Interface
- 📱 Mobile-Friendly Design
- 🔒 Secure & User-Friendly Experience

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- AI Integration
- Vercel (Deployment)

## 🚀 Live Demo

https://govt-scheme-ai.vercel.app/

## 🎯 Purpose

This project was developed to make government welfare schemes easier to discover and understand. By leveraging AI, it provides personalized recommendations based on user inputs, helping citizens quickly identify schemes they may be eligible for.
